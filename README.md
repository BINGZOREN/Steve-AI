# Steve AI - Minecraft Helper

Steve AI is your friendly Minecraft companion powered by various AI LLM providers. It provides expert help and services related to Minecraft, including gameplay tips, crafting recipes, redstone tutorials, building ideas, and more.

## Features

- **Multi-Provider Support**: Choose from OpenAI, Anthropic, and potentially other AI providers.
- **Secure API Key Input**: Enter your API key securely at startup.
- **Interactive Chat**: Engage in real-time conversations with Steve AI.
- **Minecraft Expertise**: Specialized knowledge for all things Minecraft.
- **Smooth Animations**: Enjoy a polished user interface with animations.

## How to Use

1. **Open the Application**: Open `index.html` in your web browser.
2. **Enter API Key**: Select your preferred AI provider and enter your API key.
3. **Start Chatting**: Ask Steve AI anything about Minecraft!

### Supported Providers

- **OpenAI**: Uses GPT-3.5-turbo for responses.
- **Anthropic**: Uses Claude-3-sonnet for responses.

*Note: You need valid API keys from these providers to use the application.*

## Files

- `index.html`: The main HTML structure.
- `styles.css`: CSS for styling and animations.
- `script.js`: JavaScript for functionality and API calls.
- `README.md`: This file.

## Requirements

- A modern web browser with JavaScript enabled.
- Valid API keys from supported AI providers.

## Security Note

API keys are stored temporarily in the browser's memory during the session. For production use, consider server-side implementation for better security.
